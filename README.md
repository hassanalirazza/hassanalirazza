## About Me

I am a Software Quality Assurance Engineer with close to 5 years of professional experience delivering quality across **web, mobile (iOS/Android/Kiosk), blockchain, and cloud-hosted applications**.

My work spans the full quality lifecycle — from writing structured test cases and validating REST APIs, to investigating AWS CloudWatch logs ahead of releases and owning client-facing release management end to end.

I have tested across industries including **blockchain/Web3, fintech, e-commerce, navigation, and gaming**, working in Agile/Scrum teams and directly with external client stakeholders.

Currently building this portfolio to document my skills, share reusable QA artifacts, and track my automation learning journey.

---

## Professional Summary

- 5 years of QA experience across 10+ products and multiple industries
- Designed and maintained **300+ test cases** covering functional, regression, UAT, smoke, sanity, and cross-browser scenarios
- Primary QA point of contact for clients — managing release readiness, sprint communication, and post-release validation
- Hands-on with **AWS CloudWatch and Lambda log investigation** for cloud-hosted application monitoring
- Experienced in **blockchain application testing** — MetaMask wallet, smart contract verification, token transactions, DApp wallets
- REST API validation with **Postman** — status codes, data integrity, and error handling
- Database validation using **MySQL and DBeaver** with custom SQL queries
- Load and performance testing with **JMeter**
- Automation learning with **Playwright, Selenium, and Cypress**
